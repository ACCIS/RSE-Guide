# Software Development Best Practices and Style Guide

This repository provides the 
[RSE-Guide](https://bristolcompositesinstitute.github.io/RSE-Guide/) site,
which presents a range of software development guides and best practices written for the 
Bristol Composites Institute.

The site covers a range of topics including, writing readable code, using 
version control, and writing tests to assert code is behaving correctly.
In addition, there are a number of links to the wide array of research 
[software courses](https://www.bristol.ac.uk/acrc/research-software-engineering/training/) 
provided by the 
[UoB RSE team](https://www.bristol.ac.uk/acrc/research-software-engineering/).

For any comments, issues, or suggestions for this guide, please contact 
([bci-rse@bristol.ac.uk](mailto:bci-rse@bristol.ac.uk)) or 
[raise an issue](https://github.com/BristolCompositesInstitute/RSE-Guide/issues)
in this repository on Github.
